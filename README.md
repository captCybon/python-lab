# Python Lab 🚀

The Builder's Lab workspace for experiments, prototypes, and teaching systems.

## Environment

 - Python 3.14
 - Managed with uv
 - Isolated virtual environment (.venv)

## Setup

```bash
uv python install 3.14
uv venv --python 3.14
uv sync
```

### Run a project

```
uv run python projects/space_captain/main.py
```


## Structure

```
python-lab/
   projects/
	space_captain/
   experiments/
   notebooks/
```
